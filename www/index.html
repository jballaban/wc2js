The project

The goal of this project is to build a fully functional Real Time Stratety (RTS) game engine using vanilla JavaScript/NodeJS and as a proof of concept implement an old favourite game: Warcraft 2. The rules are not to leverage any external libraries - all meaningful code should be written from scratch. The game should not leverage the DOM as much as possible and be written completely through canvas. Only latest Chrome browser is being supported at the moment. The backend will work on a serverless AWS infrastructure for state preservation, game data and multiplayer communication.

Source is available on GitHub.
